# GLX Language Support

Syntax highlighting support for **GLX (GNU Language X)**.

## Features
- `.glx` file support
- Syntax highlighting for:
  - labels (`label`, `visit`, `visible`)
  - macros (`_define_`, `_ifndef_`, `#MACRO`)
  - control flow (`if`, `switch`, `for`, `while`)
  - arrays, axis, dictionaries (`&l`, `&a`, `&d`)
  - ITER access
  - bitwise keywords

## Usage
Open any `.glx` file and start coding.

## Status
Early development / hobby language.
